# Legalize NO

### Norwegian consolidated legislation in Markdown, version-controlled with Git.

Each law is a file. Each reform is a commit.

**779 laws** · **Updated from Lovdata public data** · **NLOD 2.0**

Part of the [Legalize](https://github.com/legalize-dev/legalize) project · [legalize.dev](https://legalize.dev)

> **Early stage** — This repository is under active development. File structure, commit history, and content may undergo significant changes, including full regeneration.

## Quick start

```bash
# Clone Norwegian legislation
git clone https://github.com/legalize-dev/legalize-no.git

# Read the Constitution (1814)
cat no/LOV-1814-05-17.md

# Search for a specific section in the Penal Code
grep -A 5 "§ 1\." no/LOV-2005-05-20-28.md
```

Filenames use the official legacy identifier (e.g., `LOV-2005-05-20-28`). The normative rank (`lov`, `grunnlov`) is in each file's YAML frontmatter.

## Known limitations

**No reform history.** This repository currently contains only the current consolidated text of each law (one commit per law). Historical versions — which would allow tracking every reform as a separate git commit — require access to Lovdata's authenticated API endpoints (`/documentHistory`, `/v1/structuredRules/timeline`). Lovdata declined our request for API access on 2026-04-13. If this changes in the future, the repository will be regenerated with full version history.

**No regulations.** Only *lover* (acts/statutes) are included. Central regulations (*sentrale forskrifter*, ~3,700 files) are available in the same public data dump and may be added in a future update.
